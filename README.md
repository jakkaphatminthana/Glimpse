# Glimpse <img src="src/logo.png" alt="logo" width="22">

A minimal random image viewer. Pick a folder and shuffle through its images one at a time.

Supported formats: jpeg, png, webp.

## Features

- Choose a folder and shuffle its images randomly
- "No repeat" option — show each image once and prompt to restart when all have been shown

## Controls

- **Right arrow** / Next button — random next image
- **Left arrow** / Previous button — go back through history
- **Esc** / Stop button — return to main menu

## Development

```
npm install
npm start
```

Built on top of Electron + React + TypeScript + MUI.
